# FluxxKitExample

### Getting started

```
brew install carthage
brew install ruby
gem install bundler
```

```
bundle install --path vendor/bundle
bundle exec pod install
carthage bootstrap --platform iOS
```

1. `open FluxxKitExample.xcworkspace`
2. Run in Xcode.
